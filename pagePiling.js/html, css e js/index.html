Moura Tech

Grupo Moura dá início ao projeto de desenvolvimento de talentos na área de TI


O que é?

É uma iniciativa que tem como objetivo criar um celeiro de desenvolvimento e captação de talentos na área de Tecnologia da Informação (TI) na região para atender as necessidades de um mercado que cada dia mais necessita de profissionais com esse perfil de formação. Estimativas da Brascom mostram que para cada 7 vagas disponíveis na área de TI, há apenas 4,6 profissionais prontos para assumi-las, ou seja, faltam profissionais para atender essa demanda. Até 2024, devem ser necessárias mais 400 mil vagas para suprir essa necessidade.

Sobre o Moura Tech

Desenhado para atender as tecnologias de desenvolvimento com grande aceitação do mercado atual, tais como HTML,CSS Node.JS Express.JS React ReactNative

Com profissionais qualificados e experiência na área de desenvolvimento.
